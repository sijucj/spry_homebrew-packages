# homebrew-packages

Official Homebrew tap for Spry CLI.

## Installation

```bash
brew tap programmablemd/packages
brew install spry
```

## Usage

After installation, you can run:

```bash
spry --version
```

## Updating

To update to the latest version:

```bash
brew update
brew upgrade spry
```

## Uninstalling

To remove Spry:

```bash
brew uninstall spry
```

To remove the tap:

```bash
brew untap programmablemd/packages
```
